# VestIA - Asistente de Moda Inteligente
## ¿Qué es VestIA?
### VestIA es una aplicación web de comercio electrónico especializada en moda, potenciada por inteligencia artificial. La aplicación combina un catálogo de productos de moda con un asistente IA personalizado que ayuda a los usuarios a encontrar las prendas perfectas según sus preferencias y necesidades.
