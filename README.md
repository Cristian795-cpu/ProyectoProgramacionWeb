# VestIA - Asistente de Moda Inteligente
## ¿Qué es VestIA?
### VestIA es una aplicación web de comercio electrónico especializada en moda, potenciada por inteligencia artificial. La aplicación combina un catálogo de productos de moda con un asistente IA personalizado que ayuda a los usuarios a encontrar las prendas perfectas según sus preferencias y necesidades.

## Características principales:
### 1) Catálogo de productos de moda: Visualización de productos organizados por categorías.
### 2) Filtros inteligentes: Búsqueda por categoría, precio, talla, color y ocasión.
### 3) Carrito de compras: Gestión completa de productos seleccionados.
### 4) Asistente IA de moda: Chatbot que recomienda productos y responde preguntas de moda.
### 5) Sistema de preferencias: Perfil personalizado para guardar gustos del usuario.
### 6) Diseño responsivo: Adaptado para móviles, tablets y desktop.
